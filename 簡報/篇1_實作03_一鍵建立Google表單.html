📋 FormApp 能做什麼？

FormApp 是 GAS 內建的服務，讓你用程式碼操作 Google 表單，就像自動機器人幫你建表單。

🖱️ 手動建立表單

  1. 打開 Google Forms 網站
  2. 手動輸入表單標題
  3. 一題一題新增題目
  4. 一個一個設定選項
  5. 如果有 50 題... 😱
手動建立 50 題可能需要 1-2 小時！

⚙️ 程式自動建立

  1. 把題目寫在試算表
  2. 執行一個函式
  3. 等待 5 秒鐘
  4. 表單自動完成！
  5. 50 題也只要 5 秒 🚀
用 FormApp 只要按一次按鈕！
常見應用場景：期中考問卷、活動報名表、滿意度調查、課程評鑑、會議回饋表單

📊 步驟一：準備題目清單

我們先用 createSampleQuestions() 在試算表中建立一份範例題庫。

試算表結構（「題庫」工作表）

A 欄：題型B 欄：題目C 欄：選項（逗號分隔）
文字請輸入你的姓名
單選你最喜歡的程式語言？Python,JavaScript,Java,C++
多選你使用過哪些 Google 服務？Gmail,Drive,Docs,Sheets,Forms
下拉你的年齡區間？18以下,18-25,26-35,36-45,46以上

建立範例題庫的程式碼

// ===== 建立範例題庫 =====
function createSampleQuestions() {
  var ss = SpreadsheetApp.getActiveSpreadsheet();

  // 建立「題庫」工作表（如果不存在）
  var sheet = ss.getSheetByName('題庫');
  if (!sheet) {
    sheet = ss.insertSheet('題庫');
  }

  // 寫入標題列
  sheet.getRange('A1:C1').setValues([['題型', '題目', '選項']]);

  // 寫入範例題目
  var questions = [
    ['文字', '請輸入你的姓名', ''],
    ['單選', '你最喜歡的程式語言？', 'Python,JavaScript,Java,C++'],
    ['多選', '你使用過哪些 Google 服務？', 'Gmail,Drive,Docs,Sheets,Forms'],
    ['下拉', '你的年齡區間？', '18以下,18-25,26-35,36-45,46以上'],
    ['文字', '對這堂課有什麼建議？', ''],
    ['單選', '你會推薦這堂課嗎？', '一定會,可能會,不確定,不會']
  ];
  sheet.getRange(2, 1, questions.length, 3).setValues(questions);

  Logger.log('✅ 已建立 ' + questions.length + ' 題範例題目');
}

📋 步驟二：FormApp.create() 建立表單

用一行程式碼就能建立一個全新的 Google 表單。 

// ===== 建立表單的基本寫法 =====
function createBasicForm() {
  // 建立一個新表單，參數是表單標題
  var form = FormApp.create('我的第一個自動表單');

  // 取得表單的填寫連結
  var url = form.getPublishedUrl();
  Logger.log('📋 表單連結：' + url);

  // 取得表單的編輯連結
  var editUrl = form.getEditUrl();
  Logger.log('✏️ 編輯連結：' + editUrl);
}
小提醒：FormApp.create('標題') 會在你的 Google Drive 根目錄建立一個新表單。 回傳的 form 物件可以繼續用來新增題目和設定屬性。

程式碼解說

  • FormApp.create('標題') — 建立新表單，回傳 Form 物件
  • form.getPublishedUrl() — 取得「填寫表單」的網址（給受訪者用）
  • form.getEditUrl() — 取得「編輯表單」的網址（給管理者用）

📝 步驟三：新增文字題

文字題是最基本的題型，讓使用者輸入自由文字（姓名、建議等）。 

// ===== 新增文字題 =====
// addTextItem() — 新增一個文字輸入欄位
var textItem = form.addTextItem();
textItem.setTitle('請輸入你的姓名');
textItem.setRequired(true);  // 設為必填

// 也可以新增「段落」文字題（多行輸入）
var paragraphItem = form.addParagraphTextItem();
paragraphItem.setTitle('對這堂課有什麼建議？');
paragraphItem.setHelpText('請盡量詳細描述，謝謝！');

addTextItem()

單行文字輸入，適合短回答

  • 姓名、Email、電話
  • setRequired(true) 設必填

addParagraphTextItem()

多行文字輸入，適合長回答

  • 建議、心得、詳細描述
  • setHelpText() 加提示文字
共用方法：每種題目都可以用 .setTitle('題目')、.setRequired(true/false)、.setHelpText('提示')。

🔘 步驟四：新增選擇題

GAS 支援三種選擇題：單選、多選、下拉選單。 

// ===== 單選題（只能選一個） =====
var mcItem = form.addMultipleChoiceItem();
mcItem.setTitle('你最喜歡的程式語言？');
mcItem.setChoiceValues(['Python', 'JavaScript', 'Java', 'C++']);

// ===== 多選題（可以選多個） =====
var cbItem = form.addCheckboxItem();
cbItem.setTitle('你使用過哪些 Google 服務？');
cbItem.setChoiceValues(['Gmail', 'Drive', 'Docs', 'Sheets', 'Forms']);

// ===== 下拉選單（只能選一個，節省空間） =====
var listItem = form.addListItem();
listItem.setTitle('你的年齡區間？');
listItem.setChoiceValues(['18以下', '18-25', '26-35', '36-45', '46以上']);

單選題

addMultipleChoiceItem()

圓圈按鈕，只能選一個

多選題

addCheckboxItem()

方塊打勾，可選多個

下拉選單

addListItem()

下拉清單，選一個

⚙️ 步驟五：設定表單屬性

建立完表單和題目後，還可以設定一些進階屬性。 

// ===== 設定表單屬性 =====

// 表單描述（顯示在標題下方）
form.setDescription('這是一份自動建立的問卷，請放心填寫！');

// 送出後的確認訊息
form.setConfirmationMessage('感謝您的填寫！您的回應已成功送出 ✅');

// 收集填寫者的 Email
form.setCollectEmail(true);

// 允許修改回應
form.setAllowResponseEdits(true);

// 顯示進度列
form.setProgressBar(true);

// 隨機排列題目順序
form.setShuffleQuestions(false);
方法功能參數
setDescription()表單描述文字
setConfirmationMessage()送出後顯示的訊息文字
setCollectEmail()收集 Emailtrue / false
setAllowResponseEdits()允許修改回應true / false
setProgressBar()顯示進度列true / false

🔗 步驟六：連結回寫試算表

建立完表單後，把連結自動寫回試算表，方便查閱和分享。 

// ===== 回寫連結到試算表 =====
function writeFormLinks(form) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheet = ss.getSheetByName('題庫');

  // 在 E1:F1 寫入標題
  sheet.getRange('E1').setValue('填寫連結');
  sheet.getRange('F1').setValue('編輯連結');

  // 在 E2:F2 寫入連結
  sheet.getRange('E2').setValue(form.getPublishedUrl());
  sheet.getRange('F2').setValue(form.getEditUrl());

  // 加上超連結格式，方便點擊
  var publishedUrl = form.getPublishedUrl();
  var richText = SpreadsheetApp.newRichTextValue()
    .setText('點此填寫表單')
    .setLinkUrl(publishedUrl)
    .build();
  sheet.getRange('E3').setRichTextValue(richText);

  Logger.log('✅ 連結已回寫到試算表');
}
兩個重要連結：
📋 getPublishedUrl() — 給填寫者的連結（公開分享用）
✏️ getEditUrl() — 給管理者的連結（可以修改表單內容）

🔧 完整主程式：一鍵建立表單

把前面學到的全部組合起來，從試算表讀取題目、自動建立表單。 

// ===== 主程式：一鍵建立表單 =====
function createFormFromSheet() {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheet = ss.getSheetByName('題庫');
  var data = sheet.getDataRange().getValues();

  // 建立新表單
  var form = FormApp.create('自動建立的問卷調查');
  form.setDescription('此問卷由 GAS 自動建立');
  form.setConfirmationMessage('感謝填寫！✅');

  // 從第 2 列開始讀取題目（第 1 列是標題）
  for (var i = 1; i < data.length; i++) {
    var type = data[i][0];     // A 欄：題型
    var title = data[i][1];    // B 欄：題目
    var options = data[i][2];  // C 欄：選項

    if (type === '文字') {
      form.addTextItem().setTitle(title);
    } else if (type === '單選') {
      form.addMultipleChoiceItem()
        .setTitle(title)
        .setChoiceValues(options.split(','));
    } else if (type === '多選') {
      form.addCheckboxItem()
        .setTitle(title)
        .setChoiceValues(options.split(','));
    } else if (type === '下拉') {
      form.addListItem()
        .setTitle(title)
        .setChoiceValues(options.split(','));
    }
  }

  // 回寫連結
  sheet.getRange('E1').setValue('填寫連結');
  sheet.getRange('E2').setValue(form.getPublishedUrl());
  sheet.getRange('F1').setValue('編輯連結');
  sheet.getRange('F2').setValue(form.getEditUrl());

  Logger.log('✅ 表單建立完成：' + form.getPublishedUrl());
}

🧪 測試流程

依照以下步驟，一步一步測試你的程式。

1 先執行 createSampleQuestions()
在 GAS 編輯器上方選擇函式 → 點擊 ▶ 執行 → 等待試算表出現「題庫」工作表
2 檢查試算表
切回試算表 → 確認有「題庫」分頁 → 裡面有 6 題範例題目
3 執行 createFormFromSheet()
回到 GAS 編輯器 → 選擇此函式 → 點擊 ▶ 執行 → 第一次需授權（選擇你的帳號 → 進階 → 前往）
4 檢查結果
回到試算表 → E 欄出現「填寫連結」→ 點擊連結 → 看到自動建立的表單！
5 到 Google Drive 確認
打開 Google Drive → 在根目錄找到「自動建立的問卷調查」→ 點開確認題目正確
成功畫面：Google 表單自動產生，包含 6 道題目（文字題、單選題、多選題、下拉選單），試算表 E 欄出現可點擊的表單連結。

❓ 常見問題排解

問題原因解決方法
❌ 找不到表單 表單建立在 Drive 根目錄 到 Google Drive 首頁搜尋表單標題
❌ 授權失敗 未允許存取 Google Forms 點「進階」→「前往專案」→ 允許權限
❌ 選項為空 C 欄沒填選項或格式錯誤 確認選項用半形逗號分隔，不要有空格
❌ 題目只有一題 迴圈沒正確讀取 確認試算表從 A2 開始有資料，檢查 for 迴圈
❌ TypeError: Cannot read property 'split' 文字題的 C 欄為空觸發 split 程式碼中先判斷題型，文字題不需要 split
除錯技巧：在 GAS 編輯器中按 Ctrl+Enter 可以查看 Logger.log() 的輸出，確認每個步驟是否正確執行。也可以在「執行紀錄」中查看完整的執行結果。
