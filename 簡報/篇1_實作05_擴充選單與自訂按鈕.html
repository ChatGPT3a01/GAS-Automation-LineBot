📋 什麼是擴充選單？

Google 試算表上方有「檔案、編輯、檢視...」等選單。
GAS 可以讓你新增自己的選單，放上你寫好的功能按鈕！

🖱️ 沒有擴充選單

  • 每次要用功能，得打開 GAS 編輯器
  • 選擇函式 → 按執行
  • 同事不會用 GAS 編輯器
  • 操作麻煩，容易出錯
非技術人員無法使用你寫的功能！

🔘 有擴充選單

  • 試算表上方出現自訂選單
  • 點一下就執行功能
  • 同事也能輕鬆使用
  • 還能加上確認對話框防誤操作
任何人打開試算表就能用！
重點觀念：擴充選單讓你把試算表從「資料表格」升級成「簡易應用程式」，不需要寫前端網頁！

⚡ 步驟一：onOpen() 觸發函式

onOpen() 是 GAS 的「簡單觸發函式」，試算表每次被開啟時會自動執行。 

// ===== onOpen() — 試算表開啟時自動執行 =====
function onOpen() {
  // 取得試算表的 UI 介面
  var ui = SpreadsheetApp.getUi();

  // 建立自訂選單
  ui.createMenu('🔧 我的工具箱')       // 選單名稱
    .addItem('📊 產生報表', 'generateReport')   // 選單項目
    .addItem('➕ 新增資料', 'addNewData')
    .addSeparator()                              // 分隔線
    .addItem('🗑️ 清除資料', 'clearData')
    .addToUi();                                  // 加到介面上
}

程式碼解說

  • function onOpen() — 函式名稱必須叫 onOpen，GAS 才會自動觸發
  • SpreadsheetApp.getUi() — 取得試算表的使用者介面物件
  • ui.createMenu('名稱') — 建立一個新選單，參數是選單標題
  • .addItem('顯示文字', '函式名') — 新增一個選單項目，點擊時執行指定函式
  • .addSeparator() — 加一條分隔線，讓選單更有組織
  • .addToUi() — 最後一定要呼叫，才會把選單加到介面上
注意：函式名稱必須精確叫 onOpen（大小寫要對），否則不會自動觸發！

📋 步驟二：createMenu() 建立選單項目

選單可以有多層結構，用子選單把功能分類管理。 

// ===== 進階選單：加入子選單 =====
function onOpen() {
  var ui = SpreadsheetApp.getUi();

  // 建立子選單
  var subMenu = ui.createMenu('📂 進階功能')
    .addItem('🔄 排序資料', 'sortData')
    .addItem('📧 寄送 Email', 'sendEmail');

  // 主選單
  ui.createMenu('🔧 我的工具箱')
    .addItem('📊 產生報表', 'generateReport')
    .addItem('➕ 新增資料', 'addNewData')
    .addSeparator()
    .addSubMenu(subMenu)         // 加入子選單
    .addSeparator()
    .addItem('🗑️ 清除資料', 'clearData')
    .addToUi();
}
方法功能說明
createMenu('名稱')建立選單名稱會顯示在選單列
addItem('文字', '函式')新增項目點擊時執行指定函式
addSeparator()分隔線視覺上區隔不同功能群組
addSubMenu(子選單)子選單展開後顯示更多項目
addToUi()套用到介面最後必須呼叫

💬 步驟三：ui.alert() 確認對話框

執行重要操作前，先彈出確認對話框，避免使用者誤操作。 

// ===== 簡單的提示訊息 =====
function showMessage() {
  var ui = SpreadsheetApp.getUi();
  ui.alert('提示', '報表已經產生完成！', ui.ButtonSet.OK);
}

// ===== 確認對話框（是/否） =====
function clearData() {
  var ui = SpreadsheetApp.getUi();

  // 彈出確認對話框
  var response = ui.alert(
    '⚠️ 確認清除',                    // 標題
    '確定要清除所有資料嗎？此操作無法復原！',  // 內容
    ui.ButtonSet.YES_NO               // 按鈕：是 / 否
  );

  // 判斷使用者按了哪個按鈕
  if (response === ui.Button.YES) {
    // 使用者按了「是」→ 執行清除
    var sheet = SpreadsheetApp.getActiveSheet();
    sheet.getRange('A2:Z').clearContent();
    ui.alert('✅ 資料已清除完成！');
  } else {
    // 使用者按了「否」→ 取消操作
    ui.alert('已取消，資料沒有變動。');
  }
}

ButtonSet 按鈕類型

  • ui.ButtonSet.OK — 只有「確定」
  • ui.ButtonSet.OK_CANCEL — 確定 / 取消
  • ui.ButtonSet.YES_NO — 是 / 否
  • ui.ButtonSet.YES_NO_CANCEL — 是 / 否 / 取消

Button 回傳值

  • ui.Button.OK — 按了確定
  • ui.Button.CANCEL — 按了取消
  • ui.Button.YES — 按了是
  • ui.Button.NO — 按了否

✏️ 步驟四：ui.prompt() 輸入對話框

讓使用者在對話框中輸入文字，程式取得輸入值後繼續處理。 

// ===== 輸入對話框 =====
function addNewData() {
  var ui = SpreadsheetApp.getUi();

  // 彈出輸入對話框
  var response = ui.prompt(
    '➕ 新增資料',               // 標題
    '請輸入要新增的項目名稱：',    // 提示文字
    ui.ButtonSet.OK_CANCEL       // 按鈕
  );

  // 檢查使用者是否按了「確定」
  if (response.getSelectedButton() === ui.Button.OK) {
    // 取得使用者輸入的文字
    var inputText = response.getResponseText();

    if (inputText.trim() === '') {
      ui.alert('⚠️ 請輸入內容，不能為空！');
      return;
    }

    // 寫入試算表的最後一列
    var sheet = SpreadsheetApp.getActiveSheet();
    var lastRow = sheet.getLastRow();
    sheet.getRange(lastRow + 1, 1).setValue(inputText);
    sheet.getRange(lastRow + 1, 2).setValue(new Date());

    ui.alert('✅ 已新增：' + inputText);
  } else {
    ui.alert('已取消新增。');
  }
}

prompt 回傳物件的方法

  • response.getSelectedButton() — 取得使用者按了哪個按鈕
  • response.getResponseText() — 取得使用者輸入的文字內容

🔧 步驟五：完整應用範例

結合選單、對話框、輸入框，做一個完整的「資料管理工具」。 

// ===== 產生報表功能 =====
function generateReport() {
  var ui = SpreadsheetApp.getUi();
  var sheet = SpreadsheetApp.getActiveSheet();
  var lastRow = sheet.getLastRow();

  if (lastRow < 2) {
    ui.alert('⚠️ 目前沒有資料，無法產生報表！');
    return;
  }

  // 計算統計數據
  var dataCount = lastRow - 1;  // 扣掉標題列
  var now = Utilities.formatDate(new Date(), 'Asia/Taipei', 'yyyy/MM/dd HH:mm');

  // 在新工作表顯示報表
  var reportSheet = SpreadsheetApp.getActiveSpreadsheet()
    .getSheetByName('報表');
  if (!reportSheet) {
    reportSheet = SpreadsheetApp.getActiveSpreadsheet()
      .insertSheet('報表');
  }

  reportSheet.getRange('A1').setValue('📊 資料統計報表');
  reportSheet.getRange('A2').setValue('產生時間：' + now);
  reportSheet.getRange('A3').setValue('資料筆數：' + dataCount + ' 筆');

  ui.alert('✅ 報表已產生！請切換到「報表」工作表查看。');
}

// ===== 雙重確認的清除功能 =====
function clearDataSafe() {
  var ui = SpreadsheetApp.getUi();
  var r1 = ui.alert('⚠️ 第一次確認',
    '要清除所有資料嗎？', ui.ButtonSet.YES_NO);
  if (r1 !== ui.Button.YES) return;

  var r2 = ui.prompt('⚠️ 第二次確認',
    '請輸入「確認清除」四個字：', ui.ButtonSet.OK_CANCEL);
  if (r2.getResponseText() !== '確認清除') {
    ui.alert('輸入不正確，已取消操作。');
    return;
  }

  SpreadsheetApp.getActiveSheet().getRange('A2:Z').clearContent();
  ui.alert('✅ 資料已全部清除！');
}

🧪 測試方式

依照以下步驟測試你的擴充選單。

1 貼上程式碼
在 GAS 編輯器中貼上所有程式碼（含 onOpen 和各功能函式）
2 儲存專案
按 Ctrl + S 儲存 → 確認沒有紅色錯誤
3 重新整理試算表頁面
切到試算表 → 按 F5 或 Ctrl + R 重新整理頁面
等待幾秒鐘 → 上方選單列出現「🔧 我的工具箱」
4 測試各項功能
點擊「🔧 我的工具箱」→ 逐一測試每個選單項目
第一次執行會需要授權（選擇帳號 → 進階 → 前往）
5 驗證結果
✅ 新增資料：最後一列出現新資料
✅ 產生報表：出現「報表」工作表
✅ 清除資料：需要雙重確認後才清除
成功畫面：重新整理試算表後，上方選單列出現「🔧 我的工具箱」選單，點擊後能看到所有功能項目，執行時會彈出對話框。

❓ 常見問題排解

問題原因解決方法
❌ 選單沒出現 沒有重新整理頁面 按 F5 重新整理試算表，等幾秒讓 onOpen 執行
❌ 函式名稱不對 onOpen 拼錯或大小寫錯 確認函式名稱精確為 onOpen（O 大寫）
❌ 點選單項目沒反應 addItem 的函式名稱字串打錯 確認 addItem 第二個參數與實際函式名稱完全一致
❌ 對話框沒出現 用 GAS 編輯器直接執行 getUi() 只在試算表介面有效，需從選單觸發
❌ 權限不足 第一次執行需授權 依序操作：繼續 → 選帳號 → 進階 → 前往專案 → 允許
重要提醒：SpreadsheetApp.getUi() 只能在試算表介面中使用。如果你從 GAS 編輯器直接執行含有 getUi() 的函式，會出現錯誤訊息。請一定要從試算表的選單觸發！
