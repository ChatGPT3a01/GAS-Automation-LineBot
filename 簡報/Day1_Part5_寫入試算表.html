🎯 學習目標

完成本實作後，你將掌握以下四大核心技能：

📝 appendRow()

新增一整列資料到試算表最後一行，最直覺的寫入方式。

sheet.appendRow([值1, 值2, 值3]);

適合逐筆新增，例如即時簽到記錄。

📊 getRange().setValues()

批次寫入多列資料，一次搞定大量資料寫入。

range.setValues([[...], [...]]);

適合批量匯入，效率比迴圈高數十倍。

🕐 日期時間處理

使用 new Date() 取得當前時間，搭配 Utilities.formatDate() 格式化。

var now = new Date();

判斷簽到狀態：準時 or 遲到。

🎨 格式化試算表

設定粗體、背景色、欄寬、凍結列、框線，打造專業報表。

range.setFontWeight('bold');

讓試算表從「能用」變「好用又好看」。

🎨 格式化 API 重點解說

Google Sheets 提供豐富的格式化 API，以下是最常用的方法：

API 功能 範例
setFontWeight('bold') 粗體 headerRange.setFontWeight('bold')
setBackground('#色碼') 背景色 range.setBackground('#1565C0')
setFontColor('#色碼') 文字色 range.setFontColor('#FFFFFF')
setHorizontalAlignment('center') 置中 range.setHorizontalAlignment('center')
setColumnWidth(col, px) 欄寬 sheet.setColumnWidth(1, 60)
setFrozenRows(n) 凍結列 sheet.setFrozenRows(1)
setBorder(...) 框線 range.setBorder(true,true,true,true,true,true)

🔵 setBorder() 六個參數

range.setBorder(
  top,      // 上框線
  left,     // 左框線
  bottom,   // 下框線
  right,    // 右框線
  vertical, // 垂直內框線
  horizontal // 水平內框線
);

全部設為 true 即完整框線。

🎯 格式化的執行順序

  1. 先寫入資料（appendRow / setValues）
  2. 取得要格式化的範圍（getRange）
  3. 套用格式（可鏈式呼叫）
sheet.getRange('A1:E1')
  .setFontWeight('bold')
  .setBackground('#1565C0')
  .setFontColor('#FFFFFF');
💡 小技巧：格式化方法支援鏈式呼叫（Method Chaining），可以在同一個 range 上連續呼叫多個方法，程式碼更簡潔！

⚡ appendRow vs setValues 比較

📝 appendRow() — 逐筆新增

  • 一次新增一列資料
  • 簡單直覺，適合即時寫入
  • 自動找到最後一行
  • 適合單筆操作
// appendRow — 一次一列
sheet.appendRow([1, '小明', '準時']);
sheet.appendRow([2, '小華', '遲到']);
sheet.appendRow([3, '小美', '準時']);

3 筆資料 = 呼叫 3 次 API

📊 setValues() — 批次寫入

  • 一次寫入多列資料
  • 效率高，適合大量匯入
  • 需指定寫入範圍
  • 適合批量操作
// setValues — 一次多列（效率高！）
var data = [
  [1, '小明', '準時'],
  [2, '小華', '遲到'],
  [3, '小美', '準時']
];
sheet.getRange(2, 1, 3, 3).setValues(data);

3 筆資料 = 呼叫 1 次 API

🚀 效能差異：資料量大時，setValues 比迴圈 appendRow 快數十倍！
原因：每次 appendRow 都是一次 API 呼叫，100 筆就呼叫 100 次；setValues 只需 1 次 API 呼叫就能寫入全部資料。

getRange() 參數說明

// getRange(起始列, 起始欄, 列數, 欄數)
sheet.getRange(2, 1, 3, 3).setValues(data);
//           第2列 第1欄 共3列 共3欄

  1. 讀取資料：用 getDataRange().getValues() 取得全部簽到記錄
  2. 統計計算：遍歷每一列，統計準時與遲到人數

// 統計邏輯
var total = data.length - 1; // 扣掉標題列
var onTime = 0, late = 0;
for (var i = 1; i < data.length; i++) {
  if (data[i][3] === '準時') onTime++;
  else if (data[i][3] === '遲到') late++;
}
var onTimeRate = Math.round(onTime/total*100);

appendRow vs setValues 使用時機

appendRow() setValues()
寫入方式 逐筆新增到最後一列 批次寫入指定範圍
適用場景 即時單筆記錄（簽到、表單提交） 大量資料匯入、批次更新
效能 多筆時效能較差 多筆時效能極佳
程式碼 sheet.appendRow([...]) range.setValues([[...],[...]])
需指定範圍 不需要（自動加在最後） 需要（getRange 指定）
