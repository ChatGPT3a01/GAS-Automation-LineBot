🎯 學習目標

完成這個實作後，你將學會以下四大核心技能：

📋 1. 取得工作表

使用 SpreadsheetApp.getActiveSheet() 取得目前的工作表物件，這是所有試算表操作的起點。

var sheet = SpreadsheetApp
  .getActiveSpreadsheet()
  .getActiveSheet();

📖 2. 讀寫儲存格

使用 getRange() 指定儲存格，搭配 getValue() 讀取、setValue() 寫入資料。

sheet.getRange('A1').getValue()
sheet.getRange('A1').setValue('Hello')

⚙️ 3. 自訂函式

使用 @customfunction 標記，讓 GAS 函式可以在試算表中像 =SUM() 一樣直接呼叫。

/** @customfunction */
function calcSubtotal(qty, price) {
  return qty * price;
}

📤 4. 推播報表到 Line

統計試算表中的銷售資料，組合成格式化的報表文字，透過 Line Bot 推播通知。

sendLineNotify(token, userId,
  "📊 每日銷售報表\n💰 總金額：$2,575");

⚙️ 自訂函式 @customfunction 解說

什麼是自訂函式？

Google Apps Script 允許你建立自訂函式，讓它像內建的 =SUM()、=AVERAGE() 一樣，直接在試算表的儲存格中使用。

關鍵：加上 @customfunction 標記

/**
 * 計算小計（數量 × 單價）
 * @customfunction
 */
function calcSubtotal(quantity, price) {
  return quantity * price;
}
💡 重點！只要在 JSDoc 註解中加上 @customfunction 標記，這個函式就能在試算表中像 =SUM() 一樣使用 =calcSubtotal()。

另一個範例：等級判斷函式

/**
 * 根據金額判斷消費等級
 * @customfunction
 */
function getLevel(amount) {
  if (amount >= 1000) return '🏆 VIP';
  if (amount >= 500)  return '⭐ 高級';
  if (amount >= 200)  return '👍 一般';
  return '📌 入門';
}

在試算表中輸入 =getLevel(D2) 就能自動判斷等級！

⚠️ 限制注意：自訂函式不能使用需要授權的服務（如 MailApp、UrlFetchApp），只能做純計算。如果需要呼叫外部 API，要用其他函式搭配觸發器。

程式碼解說

function readCell() {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();

  // 取得 A1 儲存格的值
  var value = sheet.getRange('A1').getValue();

  // 組合訊息
  var message = '📖 讀取結果\n━━━━━━━━━━━━\n';
  message += 'A1 儲存格的值是：' + value;

  // 推播到 Line
  sendLineNotify(LINE_TOKEN, LINE_USER_ID, message);
}
💡 核心觀念：getRange('A1') 指定儲存格位置，.getValue() 取得該儲存格的值。這是 GAS 讀取試算表最基礎的方法！

程式碼解說

function readRange() {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();

  // 取得 A1:C5 範圍的所有值（回傳二維陣列）
  var values = sheet.getRange('A1:C5').getValues();

  var message = '📋 範圍讀取結果（A1:C5）\n━━━━━━━━━━━━\n';

  values.forEach(function(row, index) {
    message += '第' + (index + 1) + '列：' + row.join(', ') + '\n';
  });

  sendLineNotify(LINE_TOKEN, LINE_USER_ID, message);
}
💡 核心觀念：getRange('A1:C5') 指定一個範圍，.getValues()（注意有 s）回傳一個二維陣列（Array of Arrays），每一列是一個子陣列。

getValue() vs getValues()

getValue() — 取得單一儲存格的值

getValues() — 取得範圍內所有儲存格，回傳二維陣列

二維陣列結構

[
  ['品項名稱', '數量', '單價'],
  ['美式咖啡', 10, 60],
  ['拿鐵咖啡', 8, 80],
  ...
]

📚 核心 API 整理

以下是本次實作學到的所有關鍵 API，請記住這些基礎方法：

API 功能 範例
getActiveSpreadsheet() 取得目前試算表 var ss = SpreadsheetApp.getActiveSpreadsheet()
getActiveSheet() 取得工作表 var sheet = ss.getActiveSheet()
getRange() 指定儲存格或範圍 sheet.getRange('A1') 或 sheet.getRange('A1:C5')
getValue() 讀取單一儲存格的值 range.getValue()
setValue() 寫入值到儲存格 range.setValue('Hello')
getValues() 讀取範圍（回傳二維陣列） range.getValues()
getLastRow() 取得最後一列的列號 sheet.getLastRow()
💡 記憶口訣：
取表 → 指格 → 讀寫
getActiveSheet() → getRange() → getValue() / setValue()

📖 讀取流程

var sheet = SpreadsheetApp
  .getActiveSpreadsheet()
  .getActiveSheet();

// 單一值
var val = sheet.getRange('A1')
  .getValue();

// 多個值（二維陣列）
var data = sheet.getRange('A1:C5')
  .getValues();

✏️ 寫入流程

var sheet = SpreadsheetApp
  .getActiveSpreadsheet()
  .getActiveSheet();

// 寫入單一值
sheet.getRange('A1')
  .setValue('Hello');

// 寫入公式
sheet.getRange('D2')
  .setFormula('=calcSubtotal(B2,C2)');
