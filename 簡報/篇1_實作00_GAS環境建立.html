📚 什麼是 Google Apps Script？

Google Apps Script（簡稱 GAS）是 Google 提供的免費雲端程式平台，讓你可以用 JavaScript 語法操控 Google 的各種服務。

📊 試算表自動化

自動整理資料、產生報表、計算統計數字，取代重複的手動操作。

📧 自動寄信

根據試算表資料自動發送 Email，例如成績通知、活動提醒。

🤖 Line Bot 聊天機器人

串接 Line Messaging API，打造自己的聊天機器人。

🌐 Web App 網頁應用

建立 HTML 表單、網頁介面，讓別人透過網址填寫資料。

💡 為什麼選 GAS？
不用安裝軟體、不用租伺服器、不用花錢，只要有 Google 帳號就能開始。寫好的程式直接在 Google 雲端執行，24 小時不斷線。

🔎 程式碼解析：helloWorld()

function helloWorld() {
  Logger.log('Hello World！這是我的第一支 GAS 程式！');
  Logger.log('目前時間：' + new Date().toLocaleString('zh-TW', { timeZone: 'Asia/Taipei' }));
}

逐行解釋

  • function helloWorld() { — 定義一個叫做 helloWorld 的函式（function），大括號 { 表示函式的開始
  • Logger.log('Hello World！...'); — Logger.log() 是 GAS 的「印出來」指令，括號裡的文字會顯示在執行記錄中
  • new Date() — 建立一個「現在時間」的物件
  • .toLocaleString('zh-TW', { timeZone: 'Asia/Taipei' }) — 把時間轉換成台灣格式的文字（例如：2026/3/13 下午 2:30:00）
  • '目前時間：' + ... — 用 + 號把文字和時間「串接」在一起
  • } — 大括號表示函式的結束
💡 重要觀念：Logger.log() 是學 GAS 最常用的除錯工具。任何時候你想知道某個變數的值是什麼，就用 Logger.log() 印出來看。

🔎 程式碼解析：readCell()

function readCell() {
  var sheet = SpreadsheetApp.getActiveSheet();
  var value = sheet.getRange('A1').getValue();
  Logger.log('A1 的值是：' + value);
}

逐行解釋

  • var sheet = SpreadsheetApp.getActiveSheet(); — SpreadsheetApp 是 GAS 操控試算表的工具。getActiveSheet() 取得「目前正在使用的工作表」，存到變數 sheet 裡
  • sheet.getRange('A1') — 指定要操作的儲存格位置，這裡是 A1
  • .getValue() — 讀取那個儲存格的值（注意：只能讀取單一儲存格）
  • var value = ... — 把讀取到的值存到變數 value 裡
  • Logger.log('A1 的值是：' + value); — 把結果印到執行記錄
💡 SpreadsheetApp 是什麼？
它是 GAS 內建的「試算表操控工具」。就像遙控器可以控制電視一樣，SpreadsheetApp 可以控制 Google 試算表：讀取資料、寫入資料、新增工作表等等。

8-B：加入 readRange 函式

📍 位置：Apps Script 編輯器 → writeCell 函式下方

👆 動作：貼上以下程式碼，按 Ctrl+S 儲存

function readRange() {
  var sheet = SpreadsheetApp.getActiveSheet();
  var data = sheet.getRange('A1:C3').getValues();
  Logger.log('共讀取 ' + data.length + ' 列資料');
  for (var i = 0; i < data.length; i++) {
    Logger.log('第 ' + (i + 1) + ' 列：' + data[i].join(' | '));
  }
}

👆 動作：下拉選單選擇 readRange → 點擊「▶ 執行」

👀 結果：執行記錄顯示 3 列資料，每列用「|」分隔

🔎 程式碼解析：readRange()

function readRange() {
  var sheet = SpreadsheetApp.getActiveSheet();
  var data = sheet.getRange('A1:C3').getValues();
  Logger.log('共讀取 ' + data.length + ' 列資料');
  for (var i = 0; i < data.length; i++) {
    Logger.log('第 ' + (i + 1) + ' 列：' + data[i].join(' | '));
  }
}

逐行解釋

  • sheet.getRange('A1:C3') — 指定一個「範圍」，從 A1 到 C3（3 列 x 3 欄 = 9 格）
  • .getValues() — 注意是 getValues（有 s），讀取整個範圍，回傳一個「二維陣列」
  • data.length — 陣列的長度，也就是有幾列（這裡是 3）
  • for (var i = 0; i < data.length; i++) — for 迴圈，從第 0 列跑到最後一列
  • data[i] — 取得第 i 列的資料，例如 data[0] 是 ['蘋果', 10, '紅']
  • .join(' | ') — 把陣列用「 | 」串成一個字串，方便閱讀
💡 getValue() vs getValues() 差在哪？
getValue()（沒有 s）→ 讀取單一儲存格，回傳一個值
getValues()（有 s）→ 讀取整個範圍，回傳二維陣列 [[第1列], [第2列], ...]

9 批次寫入：一次寫入整個表格

📍 位置：Apps Script 編輯器 → readRange 函式下方

👆 動作：貼上以下程式碼，按 Ctrl+S 儲存

function writeRange() {
  var sheet = SpreadsheetApp.getActiveSheet();
  var data = [
    ['姓名', '分數', '等級'],
    ['王小明', 92, '優'],
    ['林小美', 85, '甲'],
    ['張大衛', 78, '乙']
  ];
  sheet.getRange(1, 7, data.length, data[0].length).setValues(data);
  Logger.log('已寫入 ' + data.length + ' 列資料到 G~I 欄！');
}

👆 動作：下拉選單選擇 writeRange → 點擊「▶ 執行」

👀 結果：切回試算表，G~I 欄出現「姓名/分數/等級」的表格

setValue() vs setValues() 比較

setValue() — 寫入單格

sheet.getRange('E1').setValue('文字');

一次只寫一個儲存格，用儲存格地址（如 'E1'）

setValues() — 批次寫入

sheet.getRange(1, 7, 4, 3).setValues(data);

一次寫入整個範圍，用 (列, 欄, 列數, 欄數) 指定位置

💡 getRange(1, 7, 4, 3) 怎麼看？
四個數字分別是：起始列=1、起始欄=7(G)、共幾列=4、共幾欄=3。所以就是 G1:I4 的範圍。

10-B：加入 calculateSum 函式

📍 位置：Apps Script 編輯器 → writeRange 函式下方

👆 動作：貼上以下程式碼，按 Ctrl+S 儲存

function calculateSum() {
  var sheet = SpreadsheetApp.getActiveSheet();
  var data = sheet.getRange('A1:A5').getValues();
  var sum = 0;
  var count = 0;
  for (var i = 0; i < data.length; i++) {
    if (typeof data[i][0] === 'number') {
      sum += data[i][0];
      count++;
    }
  }
  var avg = count > 0 ? Math.round(sum / count * 10) / 10 : 0;
  Logger.log('總和：' + sum);
  Logger.log('平均：' + avg);
  sheet.getRange('C1').setValue('總和');
  sheet.getRange('D1').setValue(sum);
  sheet.getRange('C2').setValue('平均');
  sheet.getRange('D2').setValue(avg);
  Logger.log('計算結果已寫入 C1:D2！');
}

👆 動作：下拉選單選擇 calculateSum → 點擊「▶ 執行」

👀 結果：執行記錄顯示總和 150、平均 30，試算表 C1:D2 也出現計算結果

🔎 程式碼解析：calculateSum()

var data = sheet.getRange('A1:A5').getValues();
var sum = 0;
var count = 0;
for (var i = 0; i < data.length; i++) {
  if (typeof data[i][0] === 'number') {
    sum += data[i][0];
    count++;
  }
}
var avg = count > 0 ? Math.round(sum / count * 10) / 10 : 0;

逐行解釋

  • sheet.getRange('A1:A5').getValues() — 讀取 A1 到 A5 的值，得到 [[10], [20], [30], [40], [50]]（注意：每個值都被包在一個小陣列裡）
  • var sum = 0; var count = 0; — 建立兩個變數：sum 放總和、count 放「有幾個數字」
  • typeof data[i][0] === 'number' — 檢查這格是不是數字（如果有人填了文字就跳過）
  • sum += data[i][0] — 把這格的數字加進總和（+= 是「加上去」的意思）
  • count++ — 計數器加 1，記錄找到了幾個數字
  • count > 0 ? ... : 0 — 三元運算子：如果 count 大於 0 就算平均，否則回傳 0（避免除以零錯誤）
  • Math.round(sum / count * 10) / 10 — 算平均值並四捨五入到小數點第一位
💡 data[i][0] 的 [0] 是什麼？
因為 getValues() 回傳的是二維陣列，即使只有一欄，每列的資料也會被包成 [值]。所以 data[0] 是 [10]，要取得數字 10 就要寫 data[0][0]。
