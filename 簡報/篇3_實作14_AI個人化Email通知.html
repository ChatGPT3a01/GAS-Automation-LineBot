🔍 程式碼解析：建立範例學生資料

第一個函式 createSampleStudents() 會自動在試算表中建立範例學生名單和成績。

function createSampleStudents() {
  var sheet = SpreadsheetApp.getActiveSheet();
  sheet.clear();

  var headers = ['姓名', 'Email', '國文', '英文', '數學',
                 '平均', 'AI 評語', '寄送狀態'];
  sheet.getRange(1, 1, 1, headers.length).setValues([headers]);
  sheet.getRange(1, 1, 1, headers.length)
    .setFontWeight('bold')
    .setBackground('#7B1FA2')
    .setFontColor('#FFFFFF')
    .setHorizontalAlignment('center');

  var students = [
    ['陳小明', 'your-email@gmail.com', 85, 72, 90],
    ['林小美', 'your-email@gmail.com', 92, 88, 78],
    ['張大衛', 'your-email@gmail.com', 65, 58, 45],
    ['王小華', 'your-email@gmail.com', 78, 82, 95],
    ['李建國', 'your-email@gmail.com', 55, 48, 62]
  ];

  for (var i = 0; i < students.length; i++) {
    var row = i + 2;
    sheet.getRange(row, 1, 1, 5).setValues([students[i]]);
    // F 欄：平均（公式）
    sheet.getRange(row, 6).setFormula(
      '=ROUND(AVERAGE(C' + row + ':E' + row + '),1)'
    );
  }

  sheet.setFrozenRows(1);
  Logger.log('範例學生資料建立完成！請將 Email 改成真實地址。');
}

逐行解釋

  • sheet.clear() — 清空工作表上所有資料，從頭開始
  • setValues([headers]) — 把 8 個欄位的表頭寫入第 1 列
  • .setBackground('#7B1FA2') — 設定紫色背景的表頭（呼應 AI 主題色）
  • students 陣列 — 準備 5 位範例學生的姓名、Email、三科成績
  • setFormula('=ROUND(AVERAGE(C:E),1)') — 用 Excel 公式自動計算平均分數，取到小數第一位
  • setFrozenRows(1) — 凍結表頭列，往下捲動時表頭不會消失

🔍 程式碼解析：AI 生成評語

核心函式 generateComment() 負責呼叫 Gemini AI 生成個人化鼓勵語。

function generateComment(name, chinese, english, math) {
  var url = 'https://generativelanguage.googleapis.com/v1beta/models/'
    + GEMINI_MODEL + ':generateContent?key=' + GEMINI_API_KEY;

  var avg = Math.round((chinese + english + math) / 3 * 10) / 10;

  var prompt = '你是一位溫暖的導師。請根據以下成績，'
    + '為學生寫一段 50-80 字的個人化鼓勵語。\n\n';
  prompt += '學生：' + name + '\n';
  prompt += '國文：' + chinese + ' 分\n';
  prompt += '英文：' + english + ' 分\n';
  prompt += '數學：' + math + ' 分\n';
  prompt += '平均：' + avg + ' 分\n\n';
  prompt += '要求：\n- 使用繁體中文\n- 肯定優秀科目，鼓勵弱科進步\n';
  prompt += '- 語氣溫暖正面\n- 直接回覆評語內容，不要加標題或格式';

  var payload = {
    contents: [{ parts: [{ text: prompt }] }],
    generationConfig: { temperature: 0.8, maxOutputTokens: 200 }
  };

  var options = {
    method: 'post',
    contentType: 'application/json',
    payload: JSON.stringify(payload),
    muteHttpExceptions: true
  };

  var response = UrlFetchApp.fetch(url, options);
  var json = JSON.parse(response.getContentText());

  if (json.error) {
    Logger.log('Gemini API 錯誤：' + json.error.message);
    return '（AI 評語生成失敗）';
  }

  return json.candidates[0].content.parts[0].text.trim();
}

逐行解釋

  • url = '.../' + GEMINI_MODEL + ':generateContent?key=' + GEMINI_API_KEY — 組裝 Gemini API 的網址，包含模型名稱和 API Key
  • var avg = Math.round(...) / 10 — 手動計算平均分數（取到小數第一位）
  • prompt — 給 AI 的指令：角色設定 + 學生成績 + 輸出要求
  • payload — 傳給 API 的資料結構，包含 prompt 文字和生成設定
  • UrlFetchApp.fetch(url, options) — 發送 HTTP POST 請求到 Gemini API
  • JSON.parse(response.getContentText()) — 把 API 回傳的 JSON 字串轉成物件
  • json.candidates[0].content.parts[0].text — 從回應中取出 AI 生成的評語文字

🧠 Prompt Engineering 要點

寫給 AI 的指令（prompt）品質決定了輸出的品質。來看看這個 prompt 的設計巧思：

1. 角色設定

'你是一位溫暖的導師'

告訴 AI 它的身份，讓它用合適的語氣和角度來回應。不同角色會產出截然不同的風格。

2. 具體數據

'國文：85 分\n英文：72 分\n數學：90 分'

提供完整的三科分數和平均，讓 AI 能針對個別科目做出具體的評論。

3. 明確要求

'50-80 字、繁體中文、正面鼓勵'

限制字數避免太長或太短，指定語言和風格，讓輸出更可控。

4. 創意度控制

temperature: 0.8

temperature 越高越有創意（0-2 之間）。0.8 表示「有點創意但不會太離譜」，適合寫鼓勵語。

💡 temperature 值的影響：
  • 0.1-0.3：非常精確，每次生成結果幾乎相同（適合翻譯、摘要）
  • 0.5-0.8：適度創意，有變化但不失準（適合評語、建議）
  • 1.0-1.5：高度創意，每次結果差異很大（適合故事、詩詞）
⚠️ prompt 小技巧：加上「直接回覆評語內容，不要加標題或格式」這句，可以避免 AI 回覆多餘的「以下是評語：」之類的前綴，讓輸出更乾淨。

🎨 程式碼解析：HTML Email 模板

函式 sendPersonalizedEmail() 把成績和 AI 評語包裝成漂亮的 HTML 信件。

function sendPersonalizedEmail(name, email, chinese, english, math, comment) {
  var avg = Math.round((chinese + english + math) / 3 * 10) / 10;

  var htmlBody = '<div style="font-family: Arial; max-width: 600px; ...">';
  htmlBody += '<h2 style="color: #7B1FA2; ...">📊 個人成績通知</h2>';
  htmlBody += '<p>親愛的 <strong>' + name + '</strong> 同學，你好！</p>';

  // 成績表格（條件式顏色）
  var subjects = [['國文', chinese], ['英文', english], ['數學', math]];
  for (var i = 0; i < subjects.length; i++) {
    var color = subjects[i][1] >= 80 ? '#E8F5E9'   // 綠色（≥80）
      : (subjects[i][1] >= 60 ? '#FFF3E0'             // 橙色（≥60）
      : '#FFEBEE');                                    // 紅色（<60）
    // ... 組裝表格 row
  }

  // AI 評語區
  htmlBody += '<div style="background: #F3E5F5; border-left: 4px solid #7B1FA2; ...">';
  htmlBody += '<p>💬 老師的話：</p>';
  htmlBody += '<p>' + comment + '</p></div>';

  GmailApp.sendEmail(email, name + ' 同學 — 成績通知與鼓勵', '',
    { htmlBody: htmlBody });
}

逐行解釋

  • htmlBody — 用字串拼接的方式組合 HTML 內容，包含標題、表格、評語區
  • subjects[i][1] >= 80 ? '#E8F5E9' : ... — 三元運算子：80 分以上綠色、60 分以上橙色、60 分以下紅色
  • border-left: 4px solid #7B1FA2 — AI 評語區用紫色左邊框引用樣式
  • GmailApp.sendEmail(email, 主題, '', { htmlBody }) — 第三個參數放空字串，用 htmlBody 選項寄 HTML 格式信件
💡 為什麼用 GmailApp 而不是 MailApp？GmailApp 可以設定寄件人名稱，讓信件看起來更正式。不過兩者都能寄 HTML 格式信件，功能差異不大。

🔍 程式碼解析：主流程

函式 generateAndSendAll() 是整個系統的控制中心，負責逐筆處理每位學生。

function generateAndSendAll() {
  var sheet = SpreadsheetApp.getActiveSheet();
  var lastRow = sheet.getLastRow();

  if (lastRow <= 1) {
    Logger.log('沒有學生資料，請先執行 createSampleStudents()');
    return;
  }

  var data = sheet.getRange(2, 1, lastRow - 1, 6).getValues();

  for (var i = 0; i < data.length; i++) {
    var row = i + 2;
    var name    = data[i][0];
    var email   = data[i][1];
    var chinese = data[i][2];
    var english = data[i][3];
    var math    = data[i][4];

    if (!email || email.indexOf('@') === -1) {
      sheet.getRange(row, 8).setValue('略過');
      continue;
    }

    try {
      // 1. AI 生成評語
      sheet.getRange(row, 8).setValue('生成中...');
      var comment = generateComment(name, chinese, english, math);
      sheet.getRange(row, 7).setValue(comment);

      // 2. 寄送 Email
      sheet.getRange(row, 8).setValue('寄送中...');
      sendPersonalizedEmail(name, email, chinese, english, math, comment);

      sheet.getRange(row, 8).setValue('已寄出').setBackground('#E8F5E9');
      Logger.log('完成：' + name);

      // 避免 API 過載，間隔 2 秒
      Utilities.sleep(2000);

    } catch (e) {
      sheet.getRange(row, 8).setValue('失敗').setBackground('#FFEBEE');
      Logger.log('失敗：' + name + ' - ' + e.message);
    }
  }

  Logger.log('全部處理完成！');
}

逐行解釋

  • getLastRow() — 取得最後一列的列號，判斷有多少筆資料
  • getRange(2, 1, lastRow - 1, 6).getValues() — 從第 2 列開始一次讀取所有資料（A~F 欄）
  • email.indexOf('@') === -1 — 檢查 Email 是否包含 @ 符號，沒有就跳過
  • setValue('生成中...') → setValue('寄送中...') → setValue('已寄出') — 即時更新處理狀態
  • Utilities.sleep(2000) — 每處理完一位學生暫停 2 秒，避免短時間內呼叫太多次 API
  • catch (e) — 如果某位學生處理失敗，標記「失敗」並繼續處理下一位

❓ 常見問題排解

問題原因解決方法
Gemini API 錯誤 API Key 不正確 到 aistudio.google.com/apikey 確認 Key 是否正確。如果過期或被刪除，重新點「Create API Key」產生新的。
Request failed for URL 網路問題或 API 暫時不可用 等幾分鐘再試。如果持續失敗，檢查是否有防火牆或 VPN 阻擋了 Google API 的連線。
Email 寄不出去 Gmail 每日寄信上限 免費 Gmail 帳號每天最多寄 100 封。用 MailApp.getRemainingDailyQuota() 查看剩餘額度。隔天凌晨自動重置。
AI 評語太長或太短 prompt 中的字數要求 修改 prompt 中的「50-80 字」為你想要的字數範圍。也可以調整 maxOutputTokens 來控制上限。
Email 在垃圾信件匣 程式寄出的信容易被過濾 正常現象。請收件人把你的 Gmail 加入通訊錄，或從垃圾郵件中點「不是垃圾郵件」。
API key not valid API Key 無效或被撤銷 到 Google AI Studio 重新建立一組新的 API Key，貼到程式碼中。
H 欄顯示「略過」 B 欄的 Email 是空的或沒有 @ 在 B 欄填入正確的 Email 地址，重新執行 generateAndSendAll。
每位學生評語都一樣 prompt 沒有帶入個別成績 確認 generateComment() 的參數有正確傳入每位學生的姓名和成績。
⚠️ 錯誤排查小技巧：出錯時，先看 Apps Script 編輯器下方的「執行記錄」面板。紅色的訊息就是錯誤原因，把錯誤訊息複製貼到 Google 搜尋，通常都能找到解答。
